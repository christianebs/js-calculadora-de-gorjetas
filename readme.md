<h2><img width="20px" src="https://github.com/christianebs/christianebs/assets/108686840/45469bba-2011-45d3-8d45-d03d56753e32"> Sobre o Projeto </h2>

Este projeto foi criado com o intuito de simplificar o processo de divisão de contas, como jantares, ao somar o valor total com a taxa de serviço e em seguida dividir pelo número de pessoas envolvidas na despesa.
